LangChain Search Agent

Developed a modular LLM system using **LangChain** and **GPT-4** that performs real-time research. The system doesn't just answer questions; it queries the web, summarizes findings, and adapts its prompt strategy based on confidence scores.

  • Adaptive re-prompting for document summarization.
  • PromptLayer integration for latency tracking.
  • Streamlit UI for real-time interaction.
